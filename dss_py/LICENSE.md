# dss-py

A pure Python library for DSS (Data Sheet Standard) parsing, serialization, and conversion to/from CSV/XLSX/XLS/XLSM. No dependencies.

## Usage

```python
from dss import parse_dss, serialize_dss, csv_to_dss, dss_to_csv, xlsx_to_dss, dss_to_xlsx
```
